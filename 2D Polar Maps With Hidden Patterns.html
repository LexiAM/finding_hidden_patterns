Generate Data Samples With Hidden Local Patterns¶

In semiconductor or optics industries production pipelines often perform quality control, continuous improvement or R&D investigations by sampling performance on semiconductor wafers or other substrates over 2D polar maps. Thin films used in fabrication of electronic and opto-electronic devices are sampled for thicknesses, conductivity, optical or other film properties. Resulting 2D maps are used to estimate yields or find unwanted spatial variations.

This is a helper notebook for generating 2D polar maps with various spatial trends and hidden patterns: Concave, Convex, Donut-shaped radial patterns and a hidden Smiley Kitty Face.

Requirements: Python 3.5+, numpy, pandas, scipy, matplotlib

Sampling Map¶

Below is a function that generates a data frame of polar map coordinates with a given diameter, number of sampling radii, and angular density of points. The output dataframe contains the following columns: X, Y, R, Theta.

# Generate sampling locations
def polar_map(diam, num_radii, angular_density):
    """Returns dataframe containing cartesian coordinates (X, Y) and polar coordinates (R, Theta)
       of a polar sampling map.
    
    Args:
        diam (float): outer diameter of the polar map
        num_radii (int): number of sampling radii, including center and outer edge
        angular_density (int): density of points on each radius.
                               Number of points for a give radius is
                               R0(center): 1 point
                               R1(1st radius): 1 * angular_density points
                               ...
                               Rn(nth radius): n * angular_density points
    
    Returns:
        pm (pandas.DataFrame): data frame df({'X': float, 'Y': float, 'R': float, 'Theta': float})
    
    """
    # initialize X, Y, R, Theta coordinates with center point at 0
    X = np.array([0.])
    Y = np.array([0.])
    R = np.array([0.])
    Theta = np.array([0.])

    # loop through radii and define points at each radius
    for n, radius in enumerate(np.linspace(0, diam / 2, num_radii)):
        if n == 0:
            continue   # skip center point that has already been added at initialization
        
        # define polar angles and ensure not to count 0/360deg twice
        num_points = angular_density * n  # number of sampling points on this radius
        angles = np.radians(np.linspace(0, 360 * (1 - 1 / num_points), num_points))
        
        # generate X, Y, R, Theta coordinates
        X = np.append(X, np.round(radius * np.cos(angles), 2))
        Y = np.append(Y, np.round(radius * np.sin(angles), 2))
        R = np.append(R, np.repeat(radius, num_points))
        Theta = np.append(Theta, np.round(np.degrees(angles), 2))
    
    # store results in data frame
    pm = pd.DataFrame({'X': X, 'Y': Y, 'R': R, 'Theta': Theta})
    
    return pm

Sample Patterns¶

Below is the code for generating random noise, power radially dependent, donut shaped, and smiley kitty patterns.

The overall pattern is a linear combination of random noise, radial dependence, and hidden patterns: Pattern = random_noise + radial_dependence + hidden_pattern.

To represent a realistic use case the default relative amplitudes of the underlying patterns is as follows:

  • Random noise is ~ 0.5% of the overall pattern
  • Hidden pattern provides < 2% contribution to the overall pattern

The relative amplitudes are easily adjusted by changing the ..._amplitude constants.

In [4]:
### Map
pm = polar_map(294, 13, 8)  # 625-point map

# Relative amplitudes of the contributing factors to overall pattern:
# Overall Pattern = radial dependence + random noise + hidden pattern
radial_dependence_amplitude = 1  # main contributing trend
random_noise_amplitude = 0.005  # ~2% noise level
hidden_pattern_amplitude = -0.02  # 4:1 signal/random noise ratio and <2% overall contribution 

# Random noise pattern
random_noise = np.random.rand(pm.X.shape[0]) * random_noise_amplitude

# Power radial dependence
# Pattern can become concave or convex depending on the sign of radial_dependence
radial = np.power(pm.R / 147, 2.5) * radial_dependence_amplitude

# Bell shape pattern with peak value = radial_dependence_amplitude
nd = norm.pdf(pm.R, pm.R.max() * 0.65, pm.R.max() / 4)
nd_scaled = nd / nd.max()
nd_shifted = nd_scaled - nd_scaled.min()
donut = (nd_shifted / nd_shifted.max()) * radial_dependence_amplitude

# Smiley face pattern
def face_map(row, amplitude):
    if (90 <= row['Y'] <= 100) and ((-75 <= row['X'] <= -35) or (35 <= row['X'] <= 75)): # eyes
        return amplitude
    elif 240 <= row['Theta'] <=300 and row['R'] == 61.25: # smile
        return amplitude
    elif ((49 <= row['R'] <= 110) and 
          ((8 <= row['Theta'] <=16) or (164 <= row['Theta'] <=172)
           or(344 <= row['Theta'] <=352) or (188 <= row['Theta'] <=196))):
        return amplitude
    elif abs(row['X']) < 1 and 5 <= row['Y'] <= 35:  # nose
        return amplitude
    else:
        return 0
smiley_face = pm.apply(face_map, axis=1, args=(hidden_pattern_amplitude,))
